This example will demonstrate how to add and use command line arguments
in your app.

Open the project or the form
make the .exe
create a shortcut to this .exe
doubleclick the shortcut

now open the 'Properties' of the shortcut
in the 'Target' textbox, at the end of the path, add a space and the command (/help or /? or /msg or your command [see the source code]) (ex. "C:\TheExe.exe" /?) 
doubleclick the shortcut again

You can use DOS Promt or create a .bat file aswell
